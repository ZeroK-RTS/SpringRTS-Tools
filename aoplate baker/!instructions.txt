Installation
-------------
1) install Python 3.9 https://www.python.org/downloads/release/python-394/
2) install PIL pip install pillow
3) install xNormal https://xnormal.net/
4) it may also be neccecary to install imagemagick https://imagemagick.org/script/download.php
5) change xnormal path in aobaker.py (depends on where you install xNormal)

Baking
-------
1) copy units to "units" subfolder
2) copy unit textures to "unittextures" subfolder
3) copy objects to "objects3d" subfolder
4) start aobaker.py 

When done
----------
1) copy *.lua files back to "units" in mod 
2) copy *.dds files back to "unittextures" subfolder 


Run with E:\Python27\python.exe aobaker.py
